# React Portfolio

Personal portfolio built with React 19 and Vite.

## Stack

- React
- Vite
- React DOM
- GitHub Pages

## Development

```bash
npm ci
npm run dev
```

## Production Build

```bash
npm run build
```

The app source lives in `src/`. The root `index.html` is the normal Vite entry file that mounts the React app into `#root`; it does not mean the site is a plain static HTML project.
